Earthquake Prediction Using Deep Learning with Spatiotemporal Priors

A unified deep learning framework for real-time multi-parameter earthquake estimation.

The Goal

The aim of this project is to design and implement a deep learning framework that can jointly and accurately predict magnitude, epicentral distance, azimuth, and focal depth of an earthquake event using only the first 3 seconds of raw waveform data. The model is optimized for low latency, high accuracy, and real-time deployment in Earthquake Early Warning (EEW) systems.

Motivation & Innovation

Traditional EEW systems often rely on staged pipelines and shallow regressors, leading to high latency, sensitivity to noise, and poor generalization across different regions. Our approach replaces these limitations with a unified end-to-end deep learning framework that:

  • Learns directly from raw seismic waveforms without handcrafted thresholds.
  • Provides robust multi-task predictions in a single forward pass.
  • Improves transparency and trust through SHAP values and Transformer attention maps.

Dataset & Preprocessing

This work uses seismic waveform data from Japan's K-NET and KiK-net networks, maintained by NIED. These networks provide dense three-component ground motion recordings (Z, N, E) sampled at 100 Hz. Each event comprises a 3-second window following P-wave onset, forming a $300 \times 3$ waveform matrix.

Preprocessing steps include mean removal, z-score normalization, and Butterworth bandpass filtering (0.1–20 Hz) to reduce noise and ensure signal quality across stations.

Methodology & Technical Approach

Our framework integrates multiple branches to capture complex spatiotemporal dependencies in seismic data:

  • Waveform Branch: Raw 3-channel, 3-second seismic waveforms processed using stacked Conv1D, BiLSTM, and Transformer encoders.
  • Tabular Branch: Handcrafted statistical features (peak displacement, mean amplitude, skewness, kurtosis) processed via dense layers.
  • Feature Fusion: Outputs from both branches are concatenated and passed to fully connected layers with four regression heads for magnitude, distance, azimuth, and depth.
  • Training Setup: Trained on K-NET and KiK-net datasets from Japan, with early stopping, dropout regularization, and uncertainty estimation using Monte Carlo dropout.

Results & Performance

The model was evaluated on test data and achieved strong performance across all parameters:

  • Magnitude: MAE = 0.18, $R^2$ = 0.97
  • Epicentral Distance: MAE = 5.21 km, $R^2$ = 0.94
  • Azimuth: MAE = 13.6°, $R^2$ = 0.89
  • Focal Depth: MAE = 2.7 km, $R^2$ = 0.90

Conclusion

This project presents a reliable and interpretable deep learning framework for earthquake early warning. By combining waveform and tabular features, integrating CNN, BiLSTM, and Transformer layers, and providing multi-task predictions, the system achieves low latency, high accuracy, and strong generalization. It holds promise for deployment in real-world EEW infrastructures, enabling timely and trustworthy alerts to save lives and reduce disaster impact.
